# tableau-de-bord-navigateur-rapide
Tableau de bord minimaliste pour navigateur, conçu pour un chargement instantané. Architecture 100 % statique (HTML / CSS / JavaScript natif), sans dépendance externe.<br/>
<ul>
<li>Recherche multi-moteurs (Qwant, DuckDuckGo, Google)</li>
<li>Gestion des favoris avec icônes</li>
<li>Fond d’écran personnalisable</li>
<li>Météo (Open-Meteo)</li>
<li>Horloge temps réel</li>
<li>Mise en cache via Service Worker pour démarrage ultra rapide</li>
<li>Fonctionnement hors-ligne</li>
</ul>
